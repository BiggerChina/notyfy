# notyfy - Yet another jQuery notification plugin

**notyfy** is a fork of the original [noty plugin](http://needim.github.com/noty) by [Needim ArabacÄ±](http://github.com/needim) that makes it easy to create **alert** - **success** - **error** - **warning** - **information** - b messages as an alternative the standard alert dialog. Each notification is added to a **queue** (**Optional**) *and is stylable by CSS.*
